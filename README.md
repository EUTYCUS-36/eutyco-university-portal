## 💎 What’s New in This Version?
Based on extensive user feedback and UX auditing, this version introduces several "Pro-Level" enhancements:

* **Premium Typography:** Transitioned to a high-legibility modern sans-serif stack (Inter/Outfit) using CSS `clamp()` for perfect readability on everything from a Lenovo laptop to a smartphone.
* **Intelligent Sidebar:** A fully responsive, collapsible sidebar that switches to an "Icons-Only" view on large screens to maximize your workspace.
* **Advanced Dark Mode:** A meticulously crafted dark theme with AA-level contrast ratios to prevent eye strain during late-night study sessions.
* **Motivational To-Do List:** More than just a list—it's a productivity hub that encourages research, reading, and networking.
* **Surgical UX Fixes:** Resolved all element overlapping, optimized card spacing for mobile, and implemented smooth hover animations that won't clip.

## 🛠️ Core Features
* **Real-Time Dashboard:** A high-level overview of GPA trends, active courses, and upcoming deadlines.
* **Interactive Study Chat:** A peer-to-peer communication hub with UI hooks for future audio/video integration.
* **Smart Tutorial:** A non-intrusive, dismissible walkthrough for first-time users to ensure no student is left behind.
* **Professional Feedback System:** A dedicated channel for users to report system issues, ensuring the platform evolves with its community.
* **Dynamic Greetings:** A personalized top-bar that adapts its greeting based on the time of day and links directly to profile management.

## 🏗️ Tech Stack
* **HTML5:** Semantic architecture for SEO and accessibility.
* **CSS3:** Advanced Grid/Flexbox layouts, CSS Variables for theme management, and modern animations.
* **Vanilla JavaScript:** Clean, modular logic for state management and DOM manipulation without the overhead of heavy frameworks.

## 🗺️ Roadmap: The Integrated Ecosystem
The Undergraduate Portal is Phase 1 of a 3-part ecosystem:
1.  **Faculty Portal (Next):** The "Producer" side where instructors upload resources and manage grades.
2.  **Master’s Hub:** A specialized research and grant-tracking environment.
3.  **Unified Landing Page:** An intelligent entry point that routes users based on their academic role.
